## Using Swift Package Manager (Xcode 11+)

Vá para `File` -> `Swift Packages` -> `Add Package Dependency`, insira [Dracula repo's URL](https://github.com/rphlfc/Dracula.git). 

## Requisitos
- iOS 14
- SwiftUI
- Xcode 12
